# Svelte App README

### Pre-Requisitos

Node (v20.16.0 ou maior) ou Bun (v1.2.3 ou maior)

### Instalação

1. Clone o repositório:

```bash
git clone https://github.com/gralmeidan/interapc.git
cd interapc
```

2. Instale as dependências:

```bash
npm install
```

ou

```bash
bun install
```

### Rodando a aplicação

Inicie a aplicação:

```bash
npm run dev
```

ou

```bash
bun dev
```
